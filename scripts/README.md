# Scripts

This directory will contain scripts that are used to automate various tasks related to the project.

Currently, we have:

- `analyze_clang_tidy_fixes.py`: A script that ingests a fixes.yaml file generated by clang-tidy and produces a report of the problems identified in a clean ranked list. This is useful for prioritizing which issues to address first.
- `run_tests_gen_coverage.sh`: A script that runs the tests and generates a code coverage report. This is useful for identifying which parts of the code are not being tested and can help guide efforts to improve test coverage.
- `migrate_fers_xml.py`: A script which interactively migrates old FERS XML files to the new schema. This is useful for ensuring that all FERS XML files are up to date and compatible with the latest version of the software.
